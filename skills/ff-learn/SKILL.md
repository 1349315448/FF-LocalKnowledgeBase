# FF Learn

Classify the lesson before writing. Read [the ingest contract](references/ingest.md) only for L2 work.

## Checkpoint

- L0: no reusable value or already covered. Write nothing.
- L1: potentially reusable but not worth blocking delivery. Append a pending event with evidence.
- L2: user-requested durable memory, verified public convention, hard-to-reverse decision, or stable high-risk rule. Plan and apply a canonical ingest.

## Integrity

- Never learn from an unverified result, failed experiment, transient path, or task transcript.
- Keep current truth concise; keep history append-only in the timeline and audit log.
- Update page, router/graph relationships, audit log, and derived cache as one reviewed operation.
- Never store secrets, personal data, credentials, raw customer content, or private source excerpts.
- Re-query the learned intent after ingest and report the exact canonical artifacts changed.
