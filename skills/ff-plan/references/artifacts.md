# Resumable Artifact Contract

Maintain five durable files for large work:

- `REQUIREMENT.md`: approved observable behavior and scope.
- `PLAN.md`: evidence, constraints, interfaces, risks, and verification strategy.
- `TASKS.md`: dependency-ordered, testable results with traceability.
- `EXECUTION_LOG.md`: actual RED/GREEN/verification evidence and deviations.
- `REVIEW.md`: requirement, test, standards, quality, and residual-risk result.

Short-lived agent handoff files belong in an ignored runtime directory, not the public repository history.
